Build a modern, attractive, production-ready website for “RoadWatch” — an AI-powered road transparency and complaint reporting platform.

Tech stack:
React, Next.js, TypeScript, Tailwind CSS, Lucide React icons, Framer Motion, shadcn/ui, Recharts, Mapbox or Leaflet, Firebase/Supabase, PostgreSQL with PostGIS, Node.js API routes.

Website goal:
Help citizens monitor road quality, view contractor and budget details, report road issues, and track complaints transparently.

Create these pages:
1. Landing Page
- Hero section with title: “Monitor Roads. Track Spending. Report Issues.”
- CTA buttons: “Report Road Issue” and “Explore Road Map”
- Animated road/map visual
- Feature cards for AI detection, budget tracking, complaint routing, IoT monitoring

2. Road Map Dashboard
- Interactive map UI
- Road condition markers: Good, Moderate, Damaged, Critical
- Click road to view details
- Show road type: NH, SH, MDR, Local Road
- Show contractor name, last relaying date, sanctioned amount, spent amount

3. Complaint Reporting Page
- Upload photo/video
- Auto GPS location
- Issue type selector: pothole, crack, waterlogging, street light, drainage
- AI severity result: Low, Medium, Critical
- Auto-route complaint to correct Executive Engineer/authority
- Submit button and complaint ID generation

4. Budget Transparency Page
- Charts for sanctioned vs spent amount
- Contractor performance score
- project timeline
- repair history
- public fund source section

5. AI Chatbot Page
- Chat interface where users can ask:
  “Who is responsible for this road?”
  “When was this road last repaired?”
  “How much was spent?”
  “How to file a complaint?”
- Voice input button
- Multi-language selector

6. Complaint Tracker Page
- Search by complaint ID
- Status cards: Pending, Assigned, In Progress, Resolved
- Officer assigned
- Expected resolution date

7. Admin/Authority Dashboard
- View complaints by area
- Assign engineer
- Update complaint status
- View analytics
- Export reports
